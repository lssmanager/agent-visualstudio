# Prisma Migrations

This directory contains all database migration files generated by Prisma Migrate.

## First-time setup

```bash
# 1. Copy and fill your env
cp .env.example .env
# Set DATABASE_URL=postgresql://user:password@host:5432/agent_visualstudio

# 2. Install Prisma CLI (already in devDependencies after running npm install)
npx prisma generate

# 3. Create the database and apply all migrations
npx prisma migrate dev --name init

# 4. (Optional) Open Prisma Studio to inspect data
npx prisma studio
```

## Applying migrations in production (Coolify / Docker)

```bash
# Runs migrate deploy (no interactive prompts)
npx prisma migrate deploy
```

Add this command to your `Dockerfile` or Coolify pre-deploy hook **before** starting the API server.
